EmuLibre is a cross-platform, open-sourced, all-in-one emulator that is still in development.

As of right now, I am working as a solo dev on this project, and so I decided to reach out to the community for support, since this isn't a task that can be accomplished by myself alone. If you wish to contribute code to this repository, feel free to make a fork and open a pull request when you are done. If you find a serious security vulnerability, please do not open it up as an issue! Reach out to me through my personal email at foam-pushpin-sly@duck.com (this is a proxy email that forwards to my personal email).

Here is a list of systems I want to have supported as a starting point (I will add to this list as this repository matures):-

	- Atari 2600 (In Progress)
	- NES (Not started)
	- GB/GBC/GBA (Not Started)
	- NDS (Not started)

And here are a list of platforms I want supported:-

	- Linux
	- Windows
	- MacOS
	- Andriod
	- iOS

I hope all of this isn't too ambitious!!
